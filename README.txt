## Инструкция по запуску ACCAC

### 1. Открыть терминал

Перейдите в папку `ACCAC_delivery` любым удобным способом.
Например:

cd /путь/к/ACCAC_delivery

Если путь неизвестен, можно открыть терминал прямо в папке `ACCAC_delivery` и выполнять команды оттуда.


### 2. Запустить установку

chmod +x install_accac.sh
./install_accac.sh


### 3. Запустить программу

cd accac_lazarus
chmod +x project1
./project1

Либо 2 раза нажать на project1



## Повторный запуск программы

Если база данных уже установлена, для повторного запуска достаточно запустить project1 (например, нажав 2 раза или используя терминал)

cd /путь_к_accac_lazarus
./project1


## Если нужно заново развернуть БД

cd /путь_к_accac_sql
chmod +x run_all.sh
./run_all.sh


## Примечание

Во всех командах вместо `/путь_к_...` нужно указать реальный путь к папке на компьютере пользователя.
